Ina Cafe

Cafe website / Desktop

In this project, we've used many photos to introduce audience to the charming atmosphere of the cafe, as well as the appetizing meals and desserts. In the header, three photos automatically change using CSS keyframes. Additionally, the gallery section employs JavaScript for enhanced interactivity.

Cafe website / Mobile

Media Queries in CSS and Java Script are using for creating Mobile website.
In the media queries, I've divided the styles into three sizes for desktop, tablet, and mobile. This ensures that the size of photos, text, and other elements is adjusted appropriately for each device.
The hamburger menu is also created using JavaScript in this website.
